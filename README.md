# redis-for-developer

## 목표
-  레디스의 사용법 및 운영 방법에 대해 이론적으로 학습

## 진행방식
1. 책을 읽고 궁금한 내용들에 대한 토론방식으로 진행
2. 스터디가 끝난 이후 읽은 내용과 토론한 내용을 정리해서 pr을 제출
